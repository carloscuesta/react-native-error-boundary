These are the props that the `FallbackComponent` **receives**:

| Property   | Type       | Default | Description                         |
|------------|------------|---------|-------------------------------------|
| error      | `Error`    |         | The thrown error                    |
| resetError | `Function` |         | A function to reset the error state |
